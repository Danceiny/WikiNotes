# 误删Microsoft Help Viewer 2.x` 解决办法
1. 在ios镜像安装文件中找到 packages\Help 路径。

2. 打开控制台命令，cd到这个路径下

3. 在控制台中键入以下命令进行安装

>msiexec /i help3_vs_net.msi vs_setup=1
