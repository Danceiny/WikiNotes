# 3.为什么从{0~2^m-1}中随机选择K来重传以估计当前负载？
 第1次冲突，假设是两个节点通信；第2次冲突，可能有4个节点进行一对一的两组通信。以此类推。这种冲突检测可以做到较为合理的负载估计，但是这种幂指数增长曲线不是最好的估计方法。
