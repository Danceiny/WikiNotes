手机号+短信验证码登录，服务器端并不保存手机号，只保存手机号的md5。
登陆成功后客户端可以获取到登陆标识，客户端可以通过登陆标识保持相对长时间的访问服务器的权限。

3. 查看朋友圈消息

4. 朋友圈评论
