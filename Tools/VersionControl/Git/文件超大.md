# 移除

```
git rm -- THE_TOO_BIG_FILE
git commit --amend -CHEAD
```
执行完这步后，这个大文件将会从你的commit记录里移除，并且以后commit都将不会再把它commit了，这时候就可以git push把本地代码push到github上了。
