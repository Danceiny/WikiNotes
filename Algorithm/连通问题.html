# 快速查找算法： 

#include<stdio.h>#define N 10000main(){  int i, p, q, t, id[N];  for(i = 0; i < N; i++)  {    id[i] = i;  }  while(scanf("%d %d\n",&p,&q) == 2)  {    if(id[p] == id[q]) continue;    for(t = id[p], i =0; i < N; i++){      if (id[i] == t) {        /* code */        id[i] = id[q];      }    }    printf(" %d %d\n", p, q);  }}

性质： 对于具有N个对象，每个对象包括M个并集运算的连通性问题，快速查找算法至少要执行MN条指令。
